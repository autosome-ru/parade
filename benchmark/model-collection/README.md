This directory contains the code and Jupyter Notebooks needed to train the following models and evaluate them:

1. Optimus-5' (`optimus5.py`)
2. Optimus-5'[25] (`optimu5_25.py`)
3. FramePool (`framepool.py`)
4. MTtrans (`mttrans.py`)
5. DREAM-RNN and DREAM-Attn models (`dream_wrappers.py`)

For DREAM models to work, copy the `prixfixe` module directory from [Random Promoter DREAM Challenge repository](https://github.com/de-Boer-Lab/random-promoter-dream-challenge-2022/) into this folder.
